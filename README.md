# 🐱 Cat Player

> 一款面向 Android 的原生音乐播放器，通过 SMB 协议直连网络共享（NAS）中的音乐文件，支持本地音乐播放。

## ✨ 功能特性

### 🎵 音乐播放
- **SMB/NAS 直连** — 无需挂载系统共享，应用内直连 SMB 服务器播放音乐
- **SFTP 协议** — SSH 安全文件传输，天然支持 seek
- **WebDAV 协议** — 支持 WebDAV 服务器流式播放
- **本地音乐** — 自动扫描本地音乐库，支持所有常见音频格式
- **无缝播放 (Gapless)** — ExoPlayer 原生播放列表模式，曲间 <50ms
- **Media3 ExoPlayer** — 基于 Google 最新播放引擎，流畅稳定
- **元数据提取** — 自动读取标题、艺术家、专辑、封面、歌词
- **Hi-Res 高保真** — 自动检测设备能力，支持 24bit/192kHz 输出

### 🎨 界面设计
- **Spotify 暗色主题** — 纯黑背景 + 绿色强调色，沉浸式视觉体验
- **10种主题预设** — Spotify绿/Apple红/Tidal蓝/深紫/海洋蓝/日落橙等
- **iOS 26 风格底栏** — 胶囊悬浮导航，弹性动画，图标+文字
- **全屏播放器** — 大封面展示 + 进度条 + 歌词同步 + 手势操作
- **首页仪表盘** — 问候语、统计卡片、每日名言、最近播放、智能歌单
- **桌面 Widget** — Glance API 4×1 小部件，快捷控制

### 🎛️ 音频效果
- **30段图形均衡器** — 精细调节，10种预设模式
- **参数均衡器 (PEQ)** — 中心频率/Q值/增益独立控制
- **低音增强** — 一键开启低音增强
- **ReplayGain** — 响度标准化，自动平衡不同歌曲音量
- **声道控制** — 左右平衡滑块
- **音调独立控制** — 0.5x ~ 2.0x 音调调节，不影响速度
- **音频可视化** — FFT 实时频谱条形图
- **AB 循环** — A 点到 B 点循环播放

### 📋 播放管理
- **播放队列** — 拖拽排序，滑动删除
- **多播放列表** — 命名列表 CRUD，DataStore 持久化
- **智能播放列表** — 最近添加/随机歌单/短歌曲/长歌曲
- **睡眠定时器** — 定时停止播放
- **收藏夹** — 一键收藏喜欢的歌曲
- **播放历史** — 自动记录，持久化保存
- **播放统计** — 总播放次数/时长/最常播放 Top 10
- **重复检测** — 按标题+艺术家+时长匹配
- **标签编辑** — 修改 ID3 标签（标题/艺术家/专辑）
- **文件夹浏览** — 按目录组织音乐
- **排序系统** — 按标题/歌手/专辑/时长/添加时间

### 🔤 歌词
- **逐行同步歌词** — LRC 格式解析 + 自动滚动高亮
- **逐字同步歌词** — SYLT/Enhanced LRC 逐字高亮
- **全屏歌词** — Canvas 流光动效背景 + 沉浸式体验
- **嵌入歌词** — ID3 USLT/SYLT 帧读取

### ⚙️ 其他
- **MediaSession + 通知栏** — 系统级播放控制
- **Quick Settings Tile** — 下拉通知栏快捷播放/暂停
- **SMB 多连接书签** — 保存多个服务器，快速切换
- **手势操作** — 封面左右滑切歌，上下滑调音量
- **记住密码** — SMB 连接信息自动保存
- **搜索功能** — 歌曲、专辑、艺术家全局搜索
- **数据备份** — 设置导出/导入
- **m3u 导入导出** — 播放列表文件支持

## 🏗️ 架构

```
com.example.smbplayer/
├── data/                    # 数据层
│   ├── local/              # 本地音乐扫描
│   ├── metadata/           # 元数据提取
│   ├── player/             # ExoPlayer 封装
│   ├── settings/           # 设置持久化
│   ├── smb/                # SMB 连接管理
│   ├── audio/              # 音频效果
│   ├── favorites/          # 收藏夹
│   └── lyrics/             # LRC 歌词解析
├── domain/                  # 业务层
│   ├── ConnectUseCase      # 连接逻辑
│   ├── ManagePlaylistUseCase # 播放列表管理
│   └── PlayAudioUseCase    # 播放编排
├── ui/                      # 界面层
│   ├── library/            # 音乐库首页
│   ├── player/             # 播放器页面
│   ├── connect/            # SMB 连接
│   ├── favorites/          # 收藏夹
│   ├── playlist/           # 播放队列
│   ├── settings/           # 设置页面
│   ├── navigation/         # 导航 + 底栏
│   └── theme/              # 主题配置
├── di/                      # Hilt 依赖注入
└── service/                 # 前台播放服务
```

## 🚀 编译运行

### 环境要求
- JDK 17
- Android SDK 36
- Gradle 8.9

### 编译步骤

```bash
# 克隆仓库
git clone https://github.com/ShaoCI-Hz/Cat-Player.git
cd Cat-Player

# 设置环境变量
export JAVA_HOME=/path/to/jdk17
export ANDROID_HOME=/path/to/android-sdk

# 编译 Debug APK
./gradlew assembleDebug

# APK 输出路径
# app/build/outputs/apk/debug/app-debug.apk
```

### 直接下载

前往 [Releases](https://github.com/ShaoCI-Hz/Cat-Player/releases) 下载最新 APK。
